# How to Install Git
- [Windows](#windows)
- [macOS](#macos)
- [Linux](#linux-ubuntu)

## Windows
1. Gitの公式サイト[git](https://git-scm.com/install/windows)を開く
2. `Click here to download`をクリック

## macOS
1. Open terminal
2. Install
```brew
brew install git
```
## Linux (Ubuntu)
1. Open terminal
2. Install
```bash
sudo apt install git
```
