# 基本的なコマンド
## `ls`
```bash
ls
```
```bash
ls -l
```
## `cat`
```bash
cat README.md
```
## `cd`
```bash
cd ~/ros2_ws
```
```bash
cd
```
## `grep`
```bash
grep "ros2" README.md
```
```bash
grep -n "figure" main.tex
```
## `man`
```bash
man ls
```
